# Sistem Antrian CRUD dengan MySQL

Proyek ini adalah sistem antrian berbasis web yang menggunakan PHP dan MySQL. Sistem ini memiliki fitur:
- Pengambilan antrian
- Display antrian
- Login admin dan loket
- Pengelolaan data antrian
